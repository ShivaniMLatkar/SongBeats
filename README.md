SongBeats
Audio-lyric sentiment comparing and analyzing pipeline with automated precise summary generation.

📌 Overview
SongBeats is an advanced data pipeline designed to analyze and compare the emotional dissonance or alignment between a song's audio features (tempo, energy, valence) and its lyrical sentiment. By bridging the gap between how a song sounds and what it says, SongBeats generates a precise, automated summary of the overall emotional profile.

🎯 Primary Use Case: Studying "Mob Reaction"
The core motivation of this project is to study collective human behavior and public sentiment—specifically, how crowds ("the mob") react to certain kinds of songs.
